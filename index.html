JavaScript a DOM

Učíme se manipulovat s obsahem stránky. K tomu se používá divná a tajemná věc, které se říká DOM (Document Object Model). Pomocí ní můžeš ke všem prvkům na stránce přistupovat jako k objektům a měnit jejich vlastnosti.

Prvek, se kterým chceš v JavaScriptu pracovat musíš nejprve v DOMu vybrat. To provedeš pomocí metody document.querySelector nebo jiné podobné.

Jakmile máš prvek vybraný a uložený do proměnné, můžeš mu přímo nastavovat a měnit CSS vlastnosti, přidávat nebo odebírat CSS třídy, měnit atributy nebo jeho textový obsah.

Čtvereček

Duhová nápověda

  • V HTML přidám k elementu atribut class nebo id, nebo se nemusím a budu vybírat prosté HTML elementy, ale pak ponesu následky!
  • V JavaScriptu najdu daný HTML element pomocí document.querySelector(".class") nebo document.querySelector("#id") a uložím si ho do proměnné, se kterou budu nadále pracovat
  • HTML atributy nalezeného prvku změním pomocí prvek.attribut = 'hodnota'
  • CSS vlastnosti nalezeného prvku změním pomocí prvek.style.cssVlastnost = 'hodnota'
  • Vzpomenu si, jak se jmenují CSS vlastnosti pro barvu, písmo, atd. Víceslovné názvy vlastností (ty s pomlčkou) přepíšu v JavaScriptu do tzv. camelCase. Např. z background-color v CSS udělám v JavaScriptu backgroundColor.
